# Campus Companion

Campus Companion is a React Native app (built with Expo CLI) that helps students track events, notes, and quick links. It features Drawer + Tab + Stack navigation, forms with validation, responsive layouts, and platform-specific feedback.

---

## Screens & Features

- **Home Tab**
  - Feed: Hero banner + quick links (touchable)
  - Notes: Add/edit notes with validation and saved notes list
- **Events Drawer**
  - List of upcoming events
  - EventDetails screen with custom transition and route params
- **Profile Drawer**
  - Editable profile with email & phone validation
- **Settings Drawer**
  - Toggle options arranged with Flexbox
- **Other**
  - LegacyClock class component demonstrating lifecycle methods
  - Responsive UI with grid changes based on width
  - Platform-specific feedback (Toast on Android, Alert on iOS)
